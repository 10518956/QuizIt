# QuizIt
A streamlined Android application built with **Kotlin** and **Jetpack Compose** (or XML) designed to provide a fast, interactive quiz experience. **QuizIt** focuses on clean UI, efficient data handling, and a seamless user flow from start to finish.

---

## 🚀 Features

* **Dynamic Question Loading:** Fetches or initializes a variety of questions across different categories.
* **Real-time Validation:** Instant feedback on user answers to enhance the learning experience.
* **Progress Tracking:** Visual indicators to show users how far they are in the current quiz session.
* **Sleek Dark Mode UI:** Optimized for low-light environments with a soft, modern dark color palette.
* **Responsive Design:** Fully compatible with various screen sizes and Android API levels.

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **Framework:** Android SDK
* **Architecture:** MVVM (Model-View-ViewModel) for clean separation of concerns.
* **UI:** Jetpack Compose for modern, declarative UI components.

---

## 📦 Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/QuizIt.git
    ```
2.  **Open in Android Studio:**
    Select `File > Open` and navigate to the cloned directory.
3.  **Build the Project:**
    Allow Gradle to sync and download necessary dependencies.
4.  **Run:**
    Select your emulator or physical device and click the **Run** icon.

---

## 🚦 Usage

1.  **Start Screen:** Launch the app to see the main dashboard.
2.  **Take a Quiz:** Select a category or start a random session.
3.  **Answer Questions:** Select the best option. The app validates inputs to ensure no fields are left empty.
4.  **Results:** View your final score and performance summary at the end of the session.
